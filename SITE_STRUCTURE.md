# Структура разделов сайта бара

## 1) Главная (Landing)
- **Hero-блок**
  - Название
  - CTA-кнопки: «Посмотреть меню», «Забронировать стол», «Как добраться»
- **Ключевые преимущества**
  - Кухня/бар
  - Атмосфера/музыка
  - События/караоке/диджей (если актуально)
- **Популярное из меню (preview)**
  - 6–9 карточек + ссылка на полное меню
- **Акции/Счастливые часы (preview)**
- **События ближайшей недели (preview)**
- **Галерея/интерьер (preview)**
- **Отзывы (preview)**
- **Контакты (мини-блок)**
  - Адрес, телефон, часы, кнопка «Маршрут»

## 2) Меню
- **Категории**
  - Напитки: коктейли / крепкий алкоголь / пиво / безалкогольные
  - Еда: закуски / бургеры / салаты / горячее / десерты
- **Карточка позиции**
  - Фото, состав, вес/объём, цена
  - Аллергены/острота (иконки)
- **Фильтры/поиск**
  - Вегетарианское, острое, безалкогольное, хит и т.п.

## 3) Акции
- **Список акций**
  - Условия, срок действия
- **Счастливые часы**
- **Бонусы/лояльность** (если есть)

## 4) Бронирование
- **Форма брони**
  - Дата, время, гости, имя, телефон
  - Комментарий (пожелания)
- **Правила**
  - Депозит/время удержания столика
- **Контакт для быстрой брони**
  - Телефон/мессенджеры

## 5) События
- **Календарь/афиша**
  - Концерты, DJ-сеты, тематические вечера
- **Страница события**
  - Описание, время, вход/депозит, бронь

## 6) О баре
- **История и концепция**
- **Команда/бармены** (опционально)
- **Правила заведения**
  - Возраст, дресс-код, бронь, поведение

## 7) Галерея
- **Фото интерьера**
- **Фото блюд/коктейлей**
- **Видео** (опционально)

## 8) Отзывы
- **Отзывы гостей**
- **Ссылки на площадки**
  - Google/Яндекс/Instagram и т.п.

## 9) Контакты
- **Адрес + карта**
- **Телефоны**
- **Часы работы**
- **Соцсети/мессенджеры**
- **Как добраться/парковка**

## 10) Политики (в футере)
- **Политика конфиденциальности**
- **Пользовательское соглашение** (если нужно)
- **Cookies** (опционально)

## 11) Технические разделы (опционально)
- **404**
- **Страница «Спасибо»** после формы
- **SEO**
  - Title/Description для основных страниц
